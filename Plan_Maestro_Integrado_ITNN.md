### **1\. Descripción y Zonificación del Proyecto**

El **Laboratorio Vivo** transforma el campus del ITNN en un ecosistema productivo. Basado en la distribución espacial actual:

* **Zona de Control (Líneas Rojas):** Perímetro delimitado por el **"Escudo Verde"** y las guardarrayas mecanizadas para protección civil.  
* **Zona Interior (Área Abierta):** Espacio principal destinado al **Sistema Agrosilvopastoril** (Aves \+ Árboles).  
* **Zona de Infraestructura:** Área del invernadero donde se ubica el **Maracuyá** y los módulos de **Aeroponía**.  
* **NUEVO: Vivero Integral:** Unidad de soporte transversal para la producción de planta forestal y forrajera in-situ.

### **2\. Objetivos Estratégicos Actualizados**

1. **Soberanía de Insumos (Vivero):** Dejar de depender de donaciones externas produciendo nuestras propias especies (Cocoite, Moringa, Amapa) para el proyecto.  
2. **Validación Tecnológica (Aeroponía/Maracuyá):** Demostrar la viabilidad de cultivos de alto valor e intensivos.  
3. **Seguridad Civil (Escudo Verde):** Mitigar riesgos de incendios mediante ingeniería preventiva (guardarrayas \+ barreras vivas).  
4. **Formación Dual:** Consolidar el Servicio Social como una experiencia de ingeniería aplicada.

### **3\. Matriz de Interdependencia (Relación entre Proyectos)**

*El éxito de un proyecto desbloquea el siguiente:*

* **Guardarrayas (Tractor) → Habilita → Escudo Verde:** No se puede plantar la barrera viva si no se ha limpiado mecánicamente la zona de riesgo primero.  
* **Vivero Integral → Suministra → Agrosilvopastoril y Escudo Verde:** El vivero germina las plantas (Moringa, Cocoite) que luego serán trasplantadas a las 7 hectáreas y al perímetro.  
* **Limpieza de Tambos → Habilita → Aeroponía:** La sanidad inicial de los contenedores es crítica para evitar patógenos (Pythium) en el sistema de recirculación.

### **6\. Plataforma Tecnológica de Gestión (Dashboard)**

Para el seguimiento de este Plan Maestro, se ha desarrollado una plataforma web interactiva con las siguientes características técnicas actuales:

*   **Arquitectura:** Sistema Web basado en **Laravel 12 (PHP)** y MySQL (Framework MVC).
*   **Funcionalidad:**
    *   **Visualización:** Diagramas de Gantt dinámicos por días.
    *   **Control:** Registro de avance de actividades, tiempos y **Prioridad Nivel Actividad** (Nuevo Modelo).
    *   **Persistencia:** Base de datos centralizada (MySQL) para asegurar la integridad de la información histórica.
*   **Acceso:** Disponible en la red local del laboratorio (`http://localhost/proyecto-itnn/laravel_app/public/`).

### **7\. Conclusión**

La incorporación del **Vivero Integral** y las actividades mecanizadas iniciales (**Tractor/Limpieza**) robustecen el plan. Ahora no solo se "instalan cosas", sino que se prepara el terreno (literal y figuradamente) para asegurar el éxito. La interdependencia es clara: **Sin tractor no hay seguridad para el Escudo Verde; sin limpieza no hay sanidad para la Aeroponía; y sin Vivero no hay insumos para el futuro del Sistema Agrosilvopastoril.**
